# Webhook-Tester

A simple **Webhook Tester** made in Python to test your webhooks.

---

## 🚀 Installation & Usage

### STEP 1:

```bash
git clone https://github.com/Anonymi69/Webhook-Tester.git
cd Webhook-Tester
pip install -r requirements.txt
python main.py
```
